## 動態特效
* [Codepen](https://codepen.io/):最新、最多、最雜 Demo 都在這裡，當作資料庫，是全球首屈一指的動態特效收集平台，需要什麼就在這邊找。

    * [Tutorials](https://tympanus.net/codrops/category/tutorials/):這裡的 Demo 比較會是利用已成熟的技術加入創意後所發展的動態特效，也是上述提到有詳細說明作法的。
    * [Playground](https://tympanus.net/codrops/category/playground/):這裡的 Demo 比較會是以創新為主，可能較不去考慮支援度與效能的動態特效，所以也都相對較誇張一點。
* [NavNav+](https://navnav.co/):一開始專門以 Nav 的形式的 Demo 為主，來這邊可以找到許多 Nav 不同做法。近年開始放入許多不同類型的動態特效，讓整個網站資料更為豐富。

### GIF動圖製作
- [**gifs**](https://gifs.com/) - 在 YouTube 網址中加上字就可以馬上跳轉開始編輯，功能很多，包含字幕、裁剪、特效、色調等都可以調
- **[make a gif](https://makeagif.com/youtube-to-gif)** - 基本功能包含可上字幕、圖檔等，特點是時間可以微調到小數點後三位，很適合想要製作較精細 GIF 動圖的你
- **[imgflip](https://imgflip.com/gif-maker)** - 支援 YouTube 連結擷取；網頁很陽春，但相對來說乾淨好用；功能如上字幕、圖檔、編輯大小等基本功能都有，對於想自製 GIF 圖的你來說一定很實用
- **[gfycat](https://kikinote.net/158838#gfycat)** - 支援 YouTube 連結擷取，但也有提醒使用者容易出錯，所以還是建議下載影片後再上傳做編輯，基本功能也都有，特點是可以看到自己的 GIF 有多少瀏覽次數。
    - 不過 gfycat連結YouTube 容易出錯，大家在使用 YouTube 影片時需要多嘗試幾次，或是直接下載檔案再上傳至 gfycat 製作 GIF 動圖
- **[GIPHY](https://giphy.com/create/gifmaker)** - 算是現今最多人使用的 GIF 圖檔資料庫，裡面擁有數不盡的動圖可以下載、使用，當然也有自製 GIF 檔的功能，不過需要註冊成為會員才能使用。
    - 如果你選擇上傳影片製作 GIF 檔案，影片不能超過 60 秒，需要自己剪輯後才能上傳。且，近日 GIPHY 網站似乎有點怪怪，編輯網頁進不太去
- [**Apowersoft**](https://www.apowersoft.tw/video-to-gif-online) - 三步驟完成 GIF 動圖，只能簡單的剪出片段變成 GIF 檔案，無法加上字幕，且時間擷取最低只能以秒為單位，細節無法顧到，優點就是可以下載電腦版做更進一步的編輯
- **[imgur](https://imgur.com/upload?beta)** - 將你編輯好的影片上傳轉成 GIF 動圖，直接發布在廣大的 imgur 社群裡，讓大家都可以馬上看到你的作品。
    - imgur 無法編輯影片，你必須事先將影片截成你想要的片段再上傳

### 作品收集
* [Dribbble](https://dribbble.com/) - Discover the World’s Top Designers & Creative Professionals
* [Mobbin — UI & UX design inspiration for mobile & web apps](https://mobbin.com/)：收錄超過 30 萬個來自全球最佳設計應用的 UI 庫，節省大量的 UI 和 UX 研究時間。
* [Godly｜靈感資源網站](https://godly.website/)→這個網站收錄許多高互動的網站範例，提供給想要找一些互動創作靈感的你們！
* [Media Queries](https://mediaqueri.es/) → 蒐集了歐美的 RWD 網站，除了英文外，還有義大利、德國、法國，甚至芬蘭、捷克的網站，讓你有機會一虧異國的設計。
* [S5-Style](http://bm.s5-style.com/)→日本的WEB UI彙總網站，日本的設計風格始終維持一種大氣、簡約的感覺
* [s5-style](https://bm.s5-style.com/)
    - [日本風版型 (一)](https://www.webdesignclip.com/)
    - [日本風版型 (二)](https://responsive-jp.com/)
* [RWD JP](http://responsive-jp.com/) → P 專門蒐集日本優秀的響應式網站，每日會更新 2–3 個網站
* [lapa](http://lapa.ninja/)→由一位越南設計師創立的網站，主要收集許多Landing Page的範例，上方也有分類或是以顏色搜尋的方式，能夠迅速找到不同的作品，相信要設計一頁式網站的設計師們，這會是一個相當方便的靈感來源。
* Collect UI→ 每日更新，它收集很多優秀的UI範例，左邊分類也有依照不同的功能分類，可以說是非常方便查找資料的網站。
* [📑 Pagination | The Component Gallery](https://component.gallery/)收錄許多 UI 元件的網站，可以就圖示以及說明了解每種 UI 的樣子以及應用方式，另外還有一個分類是 Design Systems，收錄了許多現成的 UI 設計系統，都可以參考看看。
